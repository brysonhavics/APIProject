 **Team Get**
----
  Getting all the teams from the Database

* **URL**

  {Base Url}/api/Team

* **Method:**
  
  Get

  
*  **URL Params**

    None

   **Required:**
    
    None

* **Data Params**

    None

* **Success Response:**
  
  * **Code:** 200 <br />
 
* **Error Response:**

  * **Code:** None <br />
    **Content:** None

* **Notes:**

  Make sure you have data stored in the Team table before getting or you'll return nothing.
