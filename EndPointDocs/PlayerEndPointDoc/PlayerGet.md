**Players Get**
----
  Get all players from the DataBase

* **URL**

  {Base Url}/api/Player

* **Method:**
  
  Get

  
*  **URL Params**

    None

   **Required:**
    
    None

* **Data Params**

   None

* **Success Response:**
  
  * **Code:** 200 <br />
 
* **Error Response:**

* **Notes:**

  Should return a list of all of the players and some details about them. If nothing comes up your table is probably empty.
