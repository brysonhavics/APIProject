**Players Delete**
----
  Removing one player from the database

* **URL**

  {Base Url}/api/Player/{id}

* **Method:**
  
  Get

  
*  **URL Params**

    `{id} = [int]`

   **Required:**
    
    None

* **Data Params**

   None

* **Success Response:**
  
  * **Code:** 200 <br />
 
* **Error Response:**

* **Notes:**

  Removes one player from the database. Put the id of the player you want to delete in the Url.
