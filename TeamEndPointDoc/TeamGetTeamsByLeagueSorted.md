  **Team Get by League Sorted**
----
  Getting all the teams from one specific league sorted by rank

* **URL**

  {Base Url}/api/Team/League/Sorted/{id}

* **Method:**
  
  Get

  
*  **URL Params**

    id: Int\

   **Required:**
    
    None

* **Data Params**

    None

* **Success Response:**
  
  * **Code:** 200 <br />
 
* **Error Response:**

  * **Code:** None <br />
    **Content:** None

* **Notes:**

  Gets all the teams in a league and then displays them by rank.
